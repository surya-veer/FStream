# Chat Aplication using Python QtNetwork

##How to run this application
###Requirements:
1. Pyhton
2. PyQt4(for GUI)

####Procedure:
1. Run "chat.py" using this command "python chat.py"
2. Create Server on some specific port no(like. 8888)
3. Invite other user :)
